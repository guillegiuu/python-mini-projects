# 🎯 Longest Common Subsequence (LCS)

🟡 Nivel: Medio

---

## 📖 Descripción

Longest Common Subsequence (LCS) es un proyecto enfocado en la implementación de programación dinámica para resolver uno de los problemas clásicos de algoritmos.

El objetivo principal es encontrar la longitud de la subsecuencia común más larga entre dos strings, optimizando el proceso mediante una grilla (matriz).

En este proyecto se trabajan conceptos como:

- Comparación de strings carácter por carácter  
- Uso de matrices (grillas)  
- Programación dinámica  
- Subproblemas y reutilización de resultados  
- Optimización de algoritmos
