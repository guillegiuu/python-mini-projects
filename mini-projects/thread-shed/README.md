# 🧵 Thread Shed

🟡 Medium (Intermediate+)

Sistema de procesamiento de datos en Python que limpia y organiza información de ventas almacenada en un string desordenado, separándola en estructuras más accesibles como listas.

## 📌 Conceptos utilizados

- Manipulación de strings
- Métodos de strings (`.split()`, `.strip()`, etc.)
- Listas
- Iteración con loops (`for`)
- Limpieza de datos (data cleaning)
- Separación de información en estructuras organizadas
