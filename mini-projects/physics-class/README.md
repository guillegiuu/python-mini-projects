# ⚛️ Getting Ready for Physics Class

Proyecto enfocado en el uso de funciones en Python aplicadas a fórmulas básicas de física.

## 📚 Temas trabajados

- Funciones (`def`)
- Parámetros y retorno (`return`)
- Conversión de temperatura (Fahrenheit ↔ Celsius)
- Cálculo de fuerza
- Aplicación de fórmulas físicas
