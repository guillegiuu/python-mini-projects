# 📚 A Sorted Tale

🟡 Nivel: Medio

---

## 📖 Descripción

A Sorted Tale es un proyecto enfocado en la implementación y comprensión de algoritmos de ordenamiento en Python.

El objetivo principal es entender cómo funcionan internamente distintos algoritmos de sorting, comparando sus enfoques, eficiencia y comportamiento.

En este proyecto se trabajan conceptos como:

- Ordenamiento de listas de datos
- Comparación entre algoritmos de sorting
- Optimización y eficiencia (Big-O)
- Enfoques recursivos y no recursivos
