# 🎯 Hacking The Fender

🔴 Nivel: Difícil

Script que simula la detección de usuarios comprometidos leyendo un archivo CSV y generando alertas mediante archivos.

---

⬅️ [Proyecto anterior](../scrabble_project/README.md)  

🏠 [Volver al menú de proyectos](https://github.com/guillegiuu/python-mini-projects/blob/main/README.md)

---

## ⚙️ Conceptos utilizados

- Manejo de archivos con `open()`
- Lectura de archivos CSV con `csv.DictReader`
- Escritura de archivos `.txt` y `.json`
- Uso de listas para almacenar datos
- Iteración sobre estructuras de datos
- Validación de claves en diccionarios
- Módulo `os` para manipulación de archivos
