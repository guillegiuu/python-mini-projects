# 🎰 Casino con historial

🟡 Nivel: Medio  

Extensión del juego de cara o cruz agregando gestión de apuestas y registro de resultados a lo largo del tiempo.

⬅️ [Volver al proyecto principal](./README.md)

---

## 🧠 Qué hace este proyecto

Este programa simula un juego de apuestas donde un jugador puede realizar múltiples jugadas y llevar un seguimiento de su saldo y resultados.

Se incorpora un historial que permite almacenar cada jugada realizada, mostrando cómo evoluciona el estado del juego.
