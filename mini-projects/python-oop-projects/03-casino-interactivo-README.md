# 🎰 Casino interactivo (POO)

🔴 Nivel: Difícil  

Simulación completa de un casino interactivo aplicando Programación Orientada a Objetos (OOP), con múltiples rondas, validaciones y manejo dinámico del estado del jugador.

⬅️ [Proyecto anterior](./02-casino-con-historial-README.md)

🏠 [Volver al proyecto principal](./README.md)

---

## 🧠 Qué hace este proyecto

Este programa permite simular un sistema de apuestas interactivo donde el jugador puede realizar múltiples jugadas, tomar decisiones en tiempo real y ver cómo evoluciona su saldo.

Se integran clases, validaciones y flujo de ejecución continuo para representar un sistema más cercano a una aplicación real.
