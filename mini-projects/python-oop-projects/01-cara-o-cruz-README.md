# 🎰 Cara o Cruz (Básico)

🟢 Nivel: Fácil  

Simulación simple de lanzamiento de moneda utilizando Programación Orientada a Objetos (OOP) en Python.

🏠 [Volver al proyecto principal](./README.md)

---

## 🧠 Qué hace este proyecto

Este programa permite simular un juego básico de cara o cruz donde un jugador puede realizar apuestas y obtener resultados aleatorios.

Se utiliza una estructura simple basada en clases para representar al jugador y la lógica del juego.

## 🧠 Qué aprendí

- Definición de clases (`class`)
- Uso del método constructor (`__init__`)
- Creación y uso de atributos
- Definición de métodos
- Encapsulamiento de lógica en objetos
- Uso de `random` para generar resultados
- Lógica condicional aplicada a juegos simples
