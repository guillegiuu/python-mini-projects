# 🌲 Wilderness Escape (Trees)

🟡 Nivel: Medio

---

## 📖 Descripción

Wilderness Escape es un proyecto enfocado en la implementación de estructuras de datos tipo árbol (Tree) para construir una historia interactiva estilo *Choose Your Own Adventure*.

El objetivo principal es permitir al usuario tomar decisiones que lo llevan por distintos caminos dentro de la historia, utilizando nodos conectados como estructura base.

En este proyecto se trabajan conceptos como:

- Representación de datos mediante árboles
- Navegación entre nodos (decisiones)
- Modelado de historias interactivas
- Uso de listas para almacenar caminos posibles
- Simulación de estructuras jerárquicas
