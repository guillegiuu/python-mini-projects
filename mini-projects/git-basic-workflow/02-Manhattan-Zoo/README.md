# 🟢 Manhattan Zoo - Meal Regimens

Este script lee un archivo de texto (`meal-regimens.txt`) con las dietas de animales y muestra la información en consola.

Simula un caso donde una aplicación necesita cargar datos desde un archivo externo y presentarlos de forma clara.

---

## 📂 Archivos

- [`show_meals.py`](./show_meals.py) → script principal  
- [`meal-regimens.txt`](./meal-regimens.txt) → archivo con las dietas  

---

## 🚀 Cómo ejecutar

```bash
python show_meals.py
```

---

## 🧪 Ejemplo de output

```
=== MANHATTAN ZOO - MEAL REGIMENS ===

Lion:
- Meat: 5kg

Elephant:
- Vegetables: 50kg

Monkey:
- Fruits: 10kg
```

---

## 🧠 Qué aprendí

- Leer archivos con open() y read()
- Mostrar datos estructurados en consola
- Separar datos (archivo .txt) de la lógica (Python)
- Simular un caso más cercano a datos reales
