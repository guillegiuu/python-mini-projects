# 🟢 Snap Rob - Disclaimer Reader

Este script lee un archivo de texto (`disclaimer.txt`) y muestra su contenido en consola.

Simula un caso simple donde una aplicación necesita cargar información externa y presentarla al usuario.

---

## 📂 Archivos

- [`show_disclaimer.py`](./show_disclaimer.py) → script principal  
- [`disclaimer.txt`](./disclaimer.txt) → archivo con el contenido a mostrar  

---

## 🚀 Cómo ejecutar

```bash
python show_disclaimer.py
```
---

## 🧪 Ejemplo de output

```
=== DISCLAIMER ===

This is a sample disclaimer.
Use this program at your own risk.
```
---

## 🧠 Qué aprendí

- Abrir archivos con open()
- Leer contenido con .read()
- Mostrar información en consola
- Separar lógica y datos (código vs archivo externo)
