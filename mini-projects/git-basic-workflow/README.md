# 🧠 Git Basic Workflow - Mini Projects

Estos mini proyectos forman parte de mi aprendizaje práctico de Git y manejo de archivos en Python.

El objetivo es entender cómo interactuar con archivos, leer contenido y mostrar información en consola, simulando pequeños casos reales.

---

## 📂 Proyectos incluidos

### 🟢 01 - Snap Rob

Script que lee un archivo de texto con un disclaimer y lo muestra en consola.

🔗 [Ver proyecto](./01-Snap-Rob)

---

### 🟢 02 - Manhattan Zoo

Script que lee un archivo con dietas de animales y muestra la información formateada en consola.

🔗 [Ver proyecto](./02-Manhattan-Zoo)

---

## 🧠 Conceptos aplicados

- Lectura de archivos (`open`, `read`)
- Manejo de strings
- Organización de proyectos en carpetas
- Simulación de workflows reales
- Estructura básica de proyectos en Python
