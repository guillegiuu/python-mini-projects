# 🌸 Blossom HashMap

🟡 Nivel: Medio

---

## 📖 Descripción

**Blossom HashMap** es una implementación de una estructura de datos tipo diccionario en Python, construida desde cero utilizando conceptos como hashing, almacenamiento de pares clave-valor y manejo de colisiones.

El objetivo del proyecto es entender cómo funcionan internamente los hashmaps, incluyendo:

- Asignación de índices mediante funciones hash  
- Almacenamiento de datos en buckets  
- Manejo de colisiones (collision handling)  
- Búsqueda y recuperación eficiente de valores  

Este proyecto permite comprender cómo funcionan estructuras fundamentales utilizadas en Python como los diccionarios (`dict`) y su importancia en el desarrollo de software.
