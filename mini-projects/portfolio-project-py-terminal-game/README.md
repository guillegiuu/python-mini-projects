# 🎮 Survival: The First Night

🟡 Nivel: Medio  

📚 Portfolio Project - Codecademy

---

## 🧠 Descripción

**Survival: The First Night** es un juego interactivo en terminal desarrollado en Python, donde el jugador debe tomar decisiones para sobrevivir su primera noche en un entorno hostil.

A lo largo de la partida, el usuario elige entre distintas acciones, recolecta recursos y enfrenta distintos resultados según sus decisiones.

---

## 🎯 Objetivos del proyecto

- Crear un juego interactivo en terminal con Python  
- Utilizar inputs del usuario para modificar el flujo del programa  
- Implementar múltiples caminos con condicionales  
- Repetir la experiencia de juego con un loop principal  

---

## 🧩 Lógica del juego

- El jugador ingresa su nombre  
- Elige una acción inicial  
- Según la decisión tomada, obtiene cierta cantidad de recursos  
- Con esos recursos, el juego determina si sobrevive o no  
- El usuario puede volver a jugar al finalizar
